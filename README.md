# Deni Avelar | Secure Software & Cybersecurity Portfolio

A custom, responsive portfolio website focused on cybersecurity, secure software, embedded systems, and mission-focused technology.

## Live website

https://dniela22.github.io/portfolio/

## Included sections

- Hero and professional brand statement
- About
- Experience
- Projects
- Skills
- Education
- LinkedIn-only contact section

## Tech stack

- HTML5
- CSS3
- Vanilla JavaScript
- GitHub Pages
- GitHub Actions

## Deploy with Git

From inside the project folder:

```bash
git init
git remote add origin https://github.com/dniela22/portfolio.git
git add .
git commit -m "feat: rebrand portfolio"
git branch -M main
git push -u origin main
```

If the repository is already connected:

```bash
git add .
git commit -m "feat: rebrand portfolio"
git push origin main
```

## Deploy through GitHub.com

1. Open the `portfolio` repository.
2. Choose **Add file → Upload files**.
3. Upload `index.html`, `styles.css`, `script.js`, and the `.github` folder.
4. Commit directly to `main`.
5. Open the **Actions** tab and wait for the deployment workflow to finish.

## Customize later

- Replace or expand project cards in `index.html`.
- Add `assets/resume.pdf` when ready.
- Update skill descriptions as experience grows.
- Add screenshots inside the `assets` folder and reference them from project cards.
